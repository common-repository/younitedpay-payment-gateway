== Installation ==

= Prerequisites =

You must create your account on [younited.com](https://younited.com).

= Configuring the plugin =

After installing the plugin, go to settings and activate the payment method.

Fill in the API keys for your account.

After you save your API keys, you\'ll have access to different settings to control what the plugin should display on the Product and Checkout pages.
We advise you to stay in \"Test\" mode until you\'re happy with your configuration and are ready to accept payments from your customers.

Once everything is properly set up, go ahead and switch to \"Live\" mode!
